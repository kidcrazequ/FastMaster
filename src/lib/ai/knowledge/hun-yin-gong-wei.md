# 婚姻宫位论

## 婚姻宫概述

八字中**日支**为婚姻宫，也称配偶宫。日干代表命主自己，日支代表配偶和婚姻状态。婚姻宫是判断婚姻质量最关键的位置。

## 十二地支在婚姻宫的含义

### 子水在日支
- 配偶聪明伶俐，善于思考，外表清秀
- 子水为桃花之一，配偶有异性缘
- 子水为阳水，配偶性格外向但内心多虑
- 子水坐正财（男命）：妻子聪慧持家
- 子水坐正官（女命）：丈夫精明能干

### 丑土在日支
- 配偶务实稳重，性格内向偏保守
- 丑为金库，配偶善于理财或积蓄
- 丑中藏己土、辛金、癸水，配偶内在丰富
- 丑土为华盖所在（巳酉丑见丑），配偶可能有宗教或学术倾向

### 寅木在日支
- 配偶有朝气、有理想、性格直率
- 寅为阳木，配偶有领导力和进取心
- 寅中藏甲木、丙火、戊土，配偶多才多艺
- 寅为驿马所在（申子辰见寅），配偶可能奔波在外

### 卯木在日支

## 婚姻宫受冲克的影响

### 婚姻宫逢冲
- **年支冲日支**：家庭长辈反对婚姻，或早年婚姻有波折
- **月支冲日支**：与父母兄弟因婚姻产生矛盾
- **时支冲日支**：子女问题影响婚姻，或晚年婚变

### 婚姻宫逢合
- **年支合日支**：长辈支持婚姻，婚姻根基稳固
- **月支合日支**：家庭关系和睦，有利婚姻
- **时支合日支**：子女有利婚姻，晚年和谐

### 婚姻宫逢刑
- 子卯刑（无礼之刑）：夫妻间缺乏尊重
- 丑未戌三刑（恃势之刑）：因权力或地位问题产生矛盾
- 寅巳申三刑（无恩之刑）：恩义不存，忘恩负义

### 婚姻宫逢害
- 暗中矛盾，表面和气但内心不满
- 影响程度小于冲但持续时间长

## 婚姻宫的十二长生状态

日支处于不同长生状态，代表婚姻宫的能量强弱：

- **长生**：婚姻充满活力，配偶有朝气
- **沐浴**：桃花之地，配偶有魅力但感情多变
- **冠带**：婚姻正在上升期，配偶事业心强
- **临官**：婚姻稳定，配偶有地位
- **帝旺**：婚姻旺盛，但物极必反需注意
- **衰**：婚姻能量下降，需要经营
- **病**：婚姻有隐患，需要关注
- **死**：婚姻宫无力，缺乏激情（不代表婚姻不好，只是平淡）
- **墓**：婚姻宫入库，感情内敛不外露
- **绝**：与配偶缘分较薄，可能晚婚
- **胎**：婚姻将有新的开始
- **养**：婚姻在培育阶段，需要耐心

## 合婚时婚姻宫互看

### 双方婚姻宫关系
| 关系 | 含义 | 评价 |
|------|------|------|
| 六合 | 感情融洽，相处默契 | 最佳 |
| 三合 | 有共同目标，方向一致 | 优秀 |
| 比和 | 性格相似，理解容易 | 良好 |
| 相生 | 一方滋养另一方 | 良好 |
| 无关系 | 互不干扰 | 中等 |
| 相克 | 有摩擦但可化解 | 需注意 |
| 相害 | 暗中不满，沟通不畅 | 不利 |
| 相刑 | 矛盾深层，需要化解 | 不利 |
| 相冲 | 正面冲突，性格对立 | 最不利 |

### 婚姻宫与配偶星的关系
- 婚姻宫坐配偶星（男命日支藏正财，女命日支藏正官）：得正缘，婚姻美满
- 婚姻宫坐偏星（男命日支藏偏财，女命日支藏七杀）：婚姻有波折，但也代表感情浓烈
- 婚姻宫坐比劫：配偶独立或有竞争者
- 婚姻宫坐食伤：配偶有才华但不安分
- 婚姻宫坐印星：配偶善良有教养
