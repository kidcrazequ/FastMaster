# 命理探源

- **作者**：袁树珊（江苏镇江人，晚清至民国时期命理学家）
- **定位**：近代命理学的集大成之作，考订了当时几乎所有命书，注重理论与实践结合
- **特点**：学术性强，考证详实，在西方科学冲击传统观念的背景下系统整理命理学
- **地位**：与《滴天髓》《子平真诠》《穷通宝鉴》并称为近现代命理学必读经典

## 总论

### 命理学之源流

命理学肇始于战国，萌芽于汉唐，成熟于宋明，至清代而集大成。

- **战国至秦汉**：五行学说兴起，纳音推命法初成。《淮南子》《白虎通》等典籍已有论命之端
- **唐代**：李虚中以年柱为主，兼参月日时，创三柱推命法。韩愈为之作墓志铭，谓其"百不失一二"
- **宋代**：徐子平变法，以日干为主，以月令定格局，创立四柱推命体系。此为命理学之大革新
- **明清**：《三命通会》《神峰通考》《滴天髓》《子平真诠》《穷通宝鉴》等经典迭出，命理学理论趋于完善
- **近代**：袁树珊承前启后，以科学态度整理古今命书，去伪存真，系统总结命理学之理论与方法

### 八字推命法的科学性探讨

袁树珊认为，命理学虽非现代实验科学，但其中蕴含深刻的自然哲学道理：

1. **天人合一**：人禀天地之气而生，出生时刻的天干地支记录了当时的五行气场
2. **阴阳消长**：四柱八字中阴阳五行的生克制化，反映了宇宙间阴阳消长的规律
3. **统计验证**：历代命书中的论断，多经过大量案例的验证，并非凭空臆断
4. **不可偏废**：命理学不宜全信，亦不宜全废。取其精华、去其糟粕，方为正道

### 袁树珊的治学态度

- **不迷信**：凡论命之法，必以理服人，不以鬼神惑人
- **不武断**：凡有疑义之处，必广引诸书互参，不妄下断语
- **重实践**：凡所言之理，必以实际命例验证，空谈不取
- **重考据**：凡引古书之说，必考其出处，辨其真伪，去伪存真

## 论日主

### 日主强弱的精确判断法

日主即日干，为八字之核心，一切推断皆以日主为基准。判断日主强弱，须从以下四方面综合衡量：

**第一，看月令（占比最重，约四成）**：
- 日主得月令者为旺，失月令者为弱
- 甲乙木生于春月（寅卯辰），得令而旺
- 丙丁火生于夏月（巳午未），得令而旺
- 庚辛金生于秋月（申酉戌），得令而旺
- 壬癸水生于冬月（亥子丑），得令而旺

**第二，看通根（约占三成）**：
- 日主在地支中有根者为有力，无根者为虚浮
- 甲木通根于寅卯，乙木通根于卯辰
- 本气根最强，中气根次之，余气根最弱

**第三，看生扶（约占两成）**：
- 有印星生身者日主增力
- 有比劫帮身者日主增力
- 印比越多、越近日主，增力越大

**第四，看克泄耗（约占一成之参考）**：
- 官杀克身者日主减力
- 食伤泄身者日主减力
- 财星耗身者日主减力

### 月令对日主的影响

月令为提纲，为八字之核心。得令与失令之差别极大：

**得令之效用**：
- 得令日主自旺，纵四柱无帮扶，亦不为弱
- 得令如得天时地利，做事事半功倍
- 得令之命，可以任财官，大运行财官运则富贵

**失令之影响**：
- 失令日主自弱，须有生扶方能任事
- 失令如逆水行舟，须得贵人相助
- 失令之命，喜印比运生扶，行财官运则力不从心

**月令量化分析**：
- 日主临月令旺地（如甲木生寅月）：旺度最高
- 日主临月令相地（如甲木生亥月）：旺度次高
- 日主临月令休地（如甲木生巳月）：旺度偏弱
- 日主临月令囚地（如甲木生申月）：旺度较弱
- 日主临月令死地（如甲木生午月）：旺度最低

### 日主在十二宫的旺衰状态

十二宫即长生、沐浴、冠带、临官、帝旺、衰、病、死、墓、绝、胎、养：

| 十二宫 | 旺衰 | 含义 |
|--------|------|------|
| 长生 | 旺 | 如初生婴儿，生机勃勃，为吉 |
| 沐浴 | 次旺 | 如幼儿沐浴，脆弱无防，多桃花 |
| 冠带 | 渐旺 | 如青年成冠，初步成才，渐有作为 |
| 临官 | 旺 | 如壮年入仕，精力充沛，可担重任 |
| 帝旺 | 极旺 | 如君临天下，旺极之态，物极必反 |
| 衰 | 渐弱 | 如年过半百，精力渐衰 |
| 病 | 弱 | 如身染疾患，力不从心 |
| 死 | 极弱 | 如气息已绝，无力回天 |
| 墓 | 入库 | 如入土安葬，收藏之意 |
| 绝 | 最弱 | 如气数已尽，荡然无存 |
| 胎 | 渐生 | 如受气结胎，新生命萌动 |
| 养 | 将生 | 如胎儿成形，蓄势待发 |

## 论用神喜忌

### 用神取法的四大原则

用神者，八字之枢纽也。用神取法，古今诸家虽有分歧，然不出以下四大原则：

**一、扶抑法**：
- 日主旺者宜抑之：用官杀克之、食伤泄之、财星耗之
- 日主弱者宜扶之：用印绶生之、比劫帮之
- 此为用神取法之基本大法，十之六七之命可由此法取用

**二、调候法**：
- 寒命宜暖：冬月生人，金寒水冷，须丙丁火调候
- 热命宜凉：夏月生人，木火炎炽，须壬癸水调候
- 调候之用，详见《穷通宝鉴》十干在十二月之取用

**三、通关法**：
- 两行相战不下者，宜以通关之神化解
- 金木交战，以水通关（金生水、水生木）
- 木土交战，以火通关（木生火、火生土）
- 水火交战，以木通关（水生木、木生火）

**四、病药法**：
- 八字有病则需药。所谓病者，格局之破坏者也
- 正官格逢伤官为病，有印制伤为药
- 财格逢劫为病，有官制劫为药
- 病重药轻则不济，药重病轻则太过

### 喜用神与忌仇神的区分

- **用神**：八字中最需要、最能平衡全局之五行
- **喜神**：生助用神之五行，为用神的保护者
- **忌神**：克制用神之五行，为八字之大忌
- **仇神**：生助忌神之五行，间接伤害用神

**举例**：
- 日主甲木弱，用神为水（印星生身），则木为喜神（帮身），土为忌神（克水），火为仇神（生土泄木）

### 真用神与假用神的辨别

- **真用神**：得令、得地、得势，有力可用者
- **假用神**：虽透干而无根，或虽有根而被克制殆尽者

辨别要点：
1. 用神须在月令中有根，方为真用
2. 用神虽透干，若被合去或被克伤，则为假用
3. 用神被多重围克而无救者，纵在月令亦假
4. 大运流年若能补救假用神，则假亦可变真

## 论格局

### 正格八种

八字格局以月令所藏为用，共分八正格：

**一、正官格**：
- 月令透正官，或月令本气为正官
- 喜财生官、印护官，忌伤官破格、七杀混杂
- 成格条件：官星清纯有力，日主能任，财印不相碍

**二、七杀格**：
- 月令透七杀，或月令本气为七杀
- 喜食神制杀、印化杀，忌财生杀旺、制化俱无
- 成格条件：杀有制化，制化得宜，日主有根

**三、正财格**：
- 月令透正财，或月令本气为正财
- 喜食伤生财、官星护财，忌比劫夺财
- 成格条件：财旺身旺，有官星护财，无比劫争夺

**四、偏财格**：
- 月令透偏财，或月令本气为偏财
- 喜食伤生财、身旺任之，忌比劫夺财
- 成格条件：身旺财旺，食伤生财，无比劫相争

**五、正印格**：
- 月令透正印，或月令本气为正印
- 喜官杀生印、日主有根，忌财星坏印
- 成格条件：印旺身旺，有官杀相生，无财来破印

**六、偏印格**：
- 月令透偏印，或月令本气为偏印
- 喜官杀生之，忌食神被夺（枭神夺食）
- 成格条件：偏印有官杀相生，无食神被夺之患

**七、食神格**：
- 月令透食神，或月令本气为食神
- 喜身旺生之、财星流通，忌偏印夺食、官星混杂
- 成格条件：食神有力生财，日主有根，无枭印夺食

**八、伤官格**：
- 月令透伤官，或月令本气为伤官
- 喜印制之、财泄之，忌见官星（伤官见官为祸百端）
- 成格条件：视日主五行而定，金水伤官喜见官，其余喜伤尽

### 特殊格局

**从格**：
- 从财格：日主极弱无根，满盘财星，从财而论
- 从官杀格：日主极弱无根，满盘官杀，从官杀而论
- 从食伤格：日主极弱，满盘食伤，从食伤而论
- 从格喜行顺从之运，忌比劫印星破格

**化格**：
- 日干与合干紧贴相合，月令有化神当令
- 甲己化土格、乙庚化金格、丙辛化水格、丁壬化木格、戊癸化火格
- 化格成立条件极严：须合化透出化神，月令支持化神

**专旺格**：
- 曲直格（木专旺）：甲乙日主生于春，地支全寅卯辰
- 炎上格（火专旺）：丙丁日主生于夏，地支全巳午未
- 稼穑格（土专旺）：戊己日主生于四季月，地支多土
- 从革格（金专旺）：庚辛日主生于秋，地支全申酉戌
- 润下格（水专旺）：壬癸日主生于冬，地支全亥子丑

### 格局成败的判断标准

**成格三要素**：
1. 格局用神有力（得令、得地、得势）
2. 格局用神不被破坏（无冲克合破）
3. 日主能承担格局（日主有根能任财官）

**败格三要素**：
1. 格局用神无力（失令、无根、被制）
2. 格局用神被严重破坏（冲克过重、合去不存）
3. 日主太弱无法承担（身弱任不起财官）

## 论婚姻

婚姻为人生大事，袁树珊对此论述甚详，从配偶星、婚姻宫、大运流年三方面综合判断。

### 男命论妻的系统方法

**第一步：看配偶星（正财）的旺衰有无**

- 正财明透、旺相有力：主妻贤美能干，助夫有力
- 正财暗藏支中：妻子为内助型，不喜抛头露面
- 正财太旺（多财）：主有多段感情或对妻不专
- 正财被克（比劫夺财）：主克妻或妻有灾病
- 正财入墓：妻子来迟，或妻性格内敛
- 八字无正财：以偏财论妻，或婚姻较迟
- 正财被合去：妻心不在己，或妻被他人所夺

**第二步：看婚姻宫（日支）的状态**

- 日支为喜用神：配偶为助力，婚姻美满
- 日支为忌仇神：配偶多拖累，婚姻有碍
- 日支逢冲：婚姻不稳定，多变动或波折
- 日支逢合：配偶善交际，但也要看合的结果
- 日支逢刑：夫妻之间有刑伤之事
- 日支坐财星：妻星入宫，夫妻缘分深厚

**第三步：看大运流年的引动**

- 大运或流年透出正财：为恋爱结婚之期
- 大运或流年冲动婚姻宫：婚姻有变化（可结婚亦可离婚）
- 大运或流年合动婚姻宫：有婚姻之喜
- 配偶星逢大运流年而旺：为得妻助力之时

**第四步：综合判断妻子的特征和婚姻质量**

- 财星旺而日主亦旺：夫妻势均力敌，婚姻质量高
- 财星旺而日主弱：妻强夫弱，男命受制于妻
- 财星弱而日主旺：夫强妻弱，妻子较为柔顺
- 财星与日主相合：有情之合，夫妻恩爱
- 财星与日主相克相战：夫妻不和，多有口舌

### 女命论夫的系统方法

**第一步：看配偶星（正官）的旺衰有无**

- 正官明透、旺相有力：主嫁贤夫，丈夫有事业成就
- 正官暗藏支中：丈夫低调务实，不善表现
- 正官太旺或官杀混杂：感情复杂，婚姻多波折
- 正官被克（伤官克官）：主克夫或夫有灾病
- 正官入墓：丈夫来迟，或丈夫沉稳内向
- 八字无正官：以七杀论夫，或婚姻较迟
- 正官被合去：夫心不在己，或丈夫被夺

**第二步：看婚姻宫（日支）的状态**

- 日支为喜用神：丈夫为助力，婚姻幸福
- 日支为忌仇神：丈夫多拖累，婚姻有碍
- 日支逢冲：婚姻不稳定，多变动
- 日支逢合：丈夫善交际，或有第三者
- 日支逢刑：夫妻之间有刑伤之事
- 日支坐官星：夫星入宫，婚姻缘分深

**第三步：看大运流年的引动**

- 大运或流年透出正官或七杀：为恋爱结婚之期
- 大运或流年冲动婚姻宫：婚姻有变化
- 大运或流年合动婚姻宫：有婚姻之喜
- 夫星逢大运流年而旺：为得夫助力之时

**第四步：综合判断丈夫的特征和婚姻质量**

- 官星旺而日主亦旺：夫妻势均力敌，婚姻质量高
- 官星旺而日主弱：夫强妻弱，女命依附于夫
- 官星弱而日主旺：妻强夫弱，女命较为独立
- 官杀混杂无清：感情纠葛不清，婚姻坎坷

### 配偶外貌的推断方法

婚姻宫（日支）所临地支，可推断配偶外貌特征：

**婚姻宫为四桃花（子午卯酉）——配偶长相好看**：
- 日支为子：配偶眉清目秀，皮肤白净，聪慧灵巧
- 日支为午：配偶热情开朗，面色红润，相貌端正
- 日支为卯：配偶清秀俊雅，身材修长，气质温和
- 日支为酉：配偶容貌出众，五官精致，注重仪表

**婚姻宫为四长生（寅申巳亥）——配偶外向活泼**：
- 日支为寅：配偶高大威猛，性格直爽，有领导气质
- 日支为申：配偶机灵活泼，口才好，善于交际
- 日支为巳：配偶精明干练，举止端庄，思维敏捷
- 日支为亥：配偶温柔善良，心胸宽广，有艺术气质

**婚姻宫为四墓库（辰戌丑未）——配偶沉稳内敛**：
- 日支为辰：配偶沉稳厚重，面相忠厚，持家有方
- 日支为戌：配偶刚毅果敢，做事认真，为人忠诚
- 日支为丑：配偶内敛寡言，勤俭持家，踏实可靠
- 日支为未：配偶温和包容，心细如发，顾家体贴

### 婚姻不顺的命理标志

**男命婚姻不顺之征兆**：

1. **日支坐财星墓库**：如甲日生于丑月或日支为丑（丑为财库），财入墓中难出，主克妻或妻多病
2. **日支坐阳刃**：如丙午日、壬子日，阳刃刚猛，主夫妻相争，婚姻不顺
3. **比劫太旺无制**：争夺财星（妻星），主有外遇或克妻
4. **财星被合走**：他干合去正财，主妻被他人夺去，或妻心不在家
5. **日支逢多次冲克**：婚姻宫反复被冲，主婚姻动荡不安，多次变动

**女命婚姻不顺之征兆**：

1. **伤官旺而无财制**：伤官直克官星（夫星），若无财来通关，主克夫或夫妻不和
2. **官杀混杂**：正官与七杀并见，夫星不纯，主感情复杂多变
3. **食伤太旺**：泄日主秀气太过，又克制官星，主婚姻不顺
4. **日支逢多次冲克**：婚姻宫反复被冲，主婚姻动荡不安
5. **印星太旺克子星**：印旺克食伤（子女星），主晚婚或婚后子嗣不顺

### 论结婚择偶时机

结婚时机之判断，须大运流年配合命局方可应验：

**男命结婚之期**：
- 大运或流年透出正财或偏财，且力量适中
- 婚姻宫被合动（如日支为卯，流年戌合卯）
- 红鸾天喜到位之年（红鸾星到流年之位）
- 财星逢长生、临官、帝旺之运

**女命结婚之期**：
- 大运或流年透出正官或七杀，且力量适中
- 婚姻宫被合动
- 红鸾天喜到位之年
- 官星逢长生、临官、帝旺之运

**红鸾天喜速查**：
- 红鸾以年支对冲之位的前一位为是
- 天喜以年支对冲之位的后一位为是
- 流年逢红鸾天喜，主有婚恋喜事

## 论大运流年与吉凶

### 大运的作用

大运每十年一换，天干管前五年，地支管后五年：

- **大运天干**：主导前五年之吉凶趋势。天干为表象，主外在环境之变化
- **大运地支**：主导后五年之吉凶趋势。地支为内在，主实际际遇之好坏
- **干支并看**：天干地支须合看，不可偏废。干支同为喜用则大吉，同为忌仇则大凶

**大运与命局作用原则**：
1. 大运天干与命局天干发生作用（生克合冲）
2. 大运地支与命局地支发生作用（生克合冲刑害）
3. 大运干支本身的关系影响力量发挥
4. 喜用运行事顺遂，忌仇运行事多阻

### 流年太岁

流年即当年之太岁，每年一换。太岁至尊，诸神煞不可犯之：

- **犯太岁之类型**：
  - 冲太岁：年支与流年支相冲，动荡不安
  - 刑太岁：年支与流年支相刑，多灾多难
  - 害太岁：年支与流年支相害，暗中不利
  - 值太岁（本命年）：年支与流年支相同，诸事小心
  - 破太岁：年支与流年支相破，有破耗之象

- **流年作用法则**：
  1. 流年天干作用于命局天干及大运天干
  2. 流年地支作用于命局地支及大运地支
  3. 流年为喜用，则该年多吉庆
  4. 流年为忌仇，则该年多凶险

### 岁运并临的特殊意义

岁运并临，即大运干支与流年干支完全相同：

- 若并临之干支为喜用神：大吉之年，喜事重重，事业有大发展
- 若并临之干支为忌仇神：大凶之年，灾祸连连，须谨慎提防
- 古书云"岁运并临，不死自己则死他人"，此说过于绝对，须以喜忌论之
- 袁树珊认为：岁运并临主大变化，吉凶须看喜忌，不可一概而论为凶

**岁运关系总结**：
1. 岁运相生为吉，主顺遂
2. 岁运相克为凶，主阻碍
3. 岁运天克地冲为反吟，主大变动
4. 岁运天合地合为伏吟，主守旧不动
5. 大运定格局层次，流年定具体事件
