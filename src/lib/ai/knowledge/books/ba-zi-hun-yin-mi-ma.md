## 第一章 结婚时间预测

### 一、大运看适婚期

大运决定人生大方向，判断适婚期的核心方法：

**男命适婚大运**：
- 大运行财星旺地：正财运或偏财运到来，适婚期
- 大运行官杀旺地：官杀生财（间接利婚），亦为适婚期
- 大运合入日支：婚姻宫被合动，有婚姻变化

**女命适婚大运**：
- 大运行官杀旺地：正官或七杀运到来，适婚期
- 大运行财星旺地：财生官杀（间接利婚），亦为适婚期
- 大运合入日支：婚姻宫被合动，有婚姻变化

### 二、流年看结婚年份

流年为一年之太岁，判断具体结婚年份的方法：

**核心方法**：
1. **配偶星出现或旺相之年**：男命流年见财星、女命流年见官杀，且与命局喜用相符
2. **婚姻宫被合之年**：流年地支与日支六合或三合，婚姻宫被合动
3. **婚姻宫被冲之年**：冲则动，婚姻宫被冲也可能结婚（但也可能离婚，需综合判断）
4. **红鸾天喜到位之年**：红鸾主婚嫁，天喜主喜事

**红鸾天喜速查表**：

| 年支 | 红鸾 | 天喜 |
|------|------|------|
| 子 | 卯 | 酉 |
| 丑 | 寅 | 申 |
| 寅 | 丑 | 未 |
| 卯 | 子 | 午 |
| 辰 | 亥 | 巳 |
| 巳 | 戌 | 辰 |
| 午 | 酉 | 卯 |
| 未 | 申 | 寅 |
| 申 | 未 | 丑 |
| 酉 | 午 | 子 |
| 戌 | 巳 | 亥 |
| 亥 | 辰 | 戌 |

5. **桃花入命之年**：流年地支为命局桃花星，异性缘旺
6. **配偶星合入日柱之年**：配偶星天干合日干或地支合日支
7. **大运流年天合地合之年**：吉事汇聚

### 三、结婚时间的综合判断

以上条件满足越多，结婚概率越高：
- 满足 1-2 条：有恋爱或感情进展
- 满足 3-4 条：结婚可能性大
- 满足 5 条以上：几乎必定有婚喜

## 第二章 早婚与晚婚

### 一、早婚的命理标志

- 配偶星在月柱透干：青年期即遇正缘
- 配偶星旺而得用：异性缘好，容易早恋早婚
- 日支坐正配偶星（男坐正财，女坐正官）：婚姻宫有正缘，易早婚
- 官印相生/财官双美：命格好，有人追求，易早婚
- 月支为桃花：青年期桃花旺盛

### 二、晚婚的命理标志

- **配偶星不显**：命局中没有配偶星或被克尽，异性缘差
- **配偶星太旺**：财多身弱（男命）或官杀混杂（女命），选择困难
- **婚姻宫受损**：日支被刑冲破害，婚姻基础不稳
- **比劫旺盛**：独立性强，不愿妥协
- **食伤旺盛**：追求完美，眼光高
- **印星旺盛**：性格内向，社交圈窄
- **华盖入命**：孤高自傲，曲高和寡
- **日支空亡**：婚姻宫虚空，配偶来迟

### 三、晚婚的年龄界定

- 现代社会：男 30 岁后、女 28 岁后算晚婚
- 命理角度：大运第二步（约 18-28 岁）若无婚姻信息，多为晚婚
- 晚婚优势：心理更成熟，选择更理性，婚姻更稳定

## 第三章 婚姻不顺的命理标志

### 一、克妻克夫的标志

**男命克妻**：
1. 比劫旺盛克财星：比肩劫财过多，克制妻星
2. 日支坐劫财（如甲寅、庚申）：婚姻宫坐劫财，直接克妻
3. 财星入墓无出路：妻星被困，妻有灾厄
4. 日支逢冲无合解：婚姻宫被冲破，根基不稳
5. 阳刃坐日支：最为严重的克妻标志

**女命克夫**：
1. 伤官旺盛克正官：伤官专克正官（夫星），伤官过旺则严重克夫
2. 日支坐伤官：婚姻宫坐伤官，直接克夫
3. 官星入墓无出路：夫星被困，夫有灾厄
4. 官杀混杂无制化：夫星不纯，感情复杂
5. 比劫旺无官杀：独立性太强，不服管束

### 二、二婚（再婚）的命理标志

**男命二婚标志**：
1. 正偏财混杂：命局中正财偏财并见且无克合制化
2. 日支被冲：婚姻宫不稳，两次以上被冲更甚
3. 比劫旺而财弱：一次财被克尽，需再遇财
4. 妻星逢合他干：妻心不专或被他人夺走
5. 大运流年伤妻星：特定运年克妻

**女命二婚标志**：
1. 官杀混杂无制化：正官七杀并见，感情纠葛
2. 日支被冲：婚姻宫不稳
3. 伤官见官无印制：伤官克制正官（夫星）
4. 食伤旺而官弱：克夫之后需再嫁
5. 夫星被合走：丈夫心不在己

### 三、婚外情的命理标志

**男命婚外情**：
- 偏财透干而旺：偏财为情人星，透干则明恋
- 命局多桃花：异性缘过旺，守不住
- 日支被合（非配偶星合）：配偶被他人吸引
- 劫财旺透：争夺之心强

**女命婚外情**：
- 七杀透干与正官并见：两夫并立
- 命局多桃花：异性缘过旺
- 日支被合（非夫星合）：自己被他人吸引
- 伤官旺而食神弱：叛逆之心强

## 第四章 婚姻质量判断

### 一、美满婚姻的标志

1. 日支坐喜用神：配偶为命主的贵人
2. 配偶星为喜用神且有力：配偶优秀且对命主有帮助
3. 日支与月支不冲不刑：婚姻环境稳定
4. 官印相生（女命）/ 财官双美（男命）：格局配合佳
5. 命局中配偶星不被合克：夫妻关系纯正

### 二、婚姻不和的标志

1. 日支坐忌神：配偶为命主的拖累
2. 配偶星为忌神：配偶给命主带来麻烦
3. 日支逢刑：夫妻有口舌或伤痛
4. 日支逢害：夫妻暗中不和
5. 比劫夺财（男命）/ 食伤制官（女命）：有第三者插足

### 三、丧偶的命理标志

1. 配偶星逢死绝之地：配偶星彻底无气
2. 日支被天克地冲：婚姻宫被彻底破坏
3. 大运流年配偶星入墓遇绝：特定年份配偶有灾
4. 命局配偶星被严重克害无救：配偶命数不佳

## 第五章 桃花运与正缘

### 一、桃花星的类型

| 桃花类型 | 特征 | 对婚姻的影响 |
|---------|------|------------|
| 正桃花 | 桃花星为喜用神 | 因正缘得好姻缘，配偶漂亮 |
| 墙内桃花 | 桃花在日支或时支 | 婚后夫妻恩爱，感情专一 |
| 墙外桃花 | 桃花在年支或月支 | 在外有异性缘，需注意分寸 |
| 桃花劫 | 桃花为忌神 | 因色破财，或感情招灾 |
| 桃花煞 | 桃花带七杀 | 感情中易有暴力或危险 |
| 滚浪桃花 | 命局桃花过多 | 感情混乱，不专一 |

### 二、正缘出现的信号

1. 流年配偶星得力且为喜用：正缘出现
2. 流年桃花入命且配偶星有力：异性缘旺且质量高
3. 红鸾天喜同到：婚喜大概率到来
4. 流年合动婚姻宫：婚姻宫被正向合动
5. 大运配偶星长生至帝旺：正缘到来的大运周期

### 三、催旺桃花运的方法

**方位催旺**：
- 桃花星为子（北方）：向北方发展
- 桃花星为午（南方）：向南方发展
- 桃花星为卯（东方）：向东方发展
- 桃花星为酉（西方）：向西方发展

**颜色催旺**：
- 用神为木：多用绿色
- 用神为火：多用红色
- 用神为土：多用黄色
- 用神为金：多用白色
- 用神为水：多用黑色/蓝色

## 第六章 大运流年与婚姻变化

### 一、结婚之年的判断

男命：
- 流年遇正财/偏财旺且合入日柱
- 大运财星得令，流年桃花或红鸾到位
- 流年冲动婚姻宫（动则变，可为结婚也可为分手）

女命：
- 流年遇正官/七杀旺且合入日柱
- 大运官杀得令，流年桃花或红鸾到位
- 流年冲动婚姻宫

### 二、离婚之年的判断

- 流年天克地冲日柱：婚姻宫和日干都被攻击
- 流年忌神伤害配偶星：配偶星被克尽
- 大运流年比劫夺财（男命）/ 伤官制官（女命）：第三者出现
- 流年冲日支且无合解：婚姻宫被冲破无救

### 三、婚姻出问题之年的判断

- 流年刑日支：口舌纷争
- 流年害日支：暗中不和
- 流年偏财/七杀旺透：有外遇信号
- 流年比劫/食伤旺：争执增多

## 第七章 化解婚姻问题的方法

### 一、五行化解法

根据命局用神选择相应的化解方式：

| 用神五行 | 化解方法 |
|---------|---------|
| 木 | 多接触植物、住所摆放绿植、穿绿色衣服、向东方发展 |
| 火 | 多运动、参加社交活动、穿红色衣服、向南方发展 |
| 土 | 稳定作息、多接触土地、穿黄色衣服、居住中心地带 |
| 金 | 佩戴金属饰品、穿白色衣服、向西方发展 |
| 水 | 多接触水（游泳/泡温泉）、穿黑蓝色、向北方发展 |

### 二、风水化解法

- **卧室布局**：避免镜子对床、房间异型多角
- **成双摆放**：摆放成对物品增强伴侣联结感
- **色调选择**：米白/浅粉色调促进包容和温馨
- **床位方向**：根据命主用神方位选择床头朝向

### 三、时空化解法

- **分居调理**：在婚灾年份暂时分居，避开冲突高峰
- **方位调整**：灾期去命局用神方位旅行或暂住
- **择吉行事**：选择吉日进行重要的婚姻活动

### 四、自身修养化解

- 婚姻不顺者宜修身养性，提升包容心
- 克夫克妻之命宜晚婚，心理更成熟
- 感情运差时不急于恋爱，等待合适的运程
- 积德行善，改善整体运势

## 第八章 现代婚姻的命理新观点

### 一、晚婚趋势的命理解读

现代社会晚婚现象普遍，从命理角度看：
- 教育延长导致结婚年龄推后
- 经济独立使女性不急于婚嫁
- 命理中的"晚婚"不再是负面标志

### 二、二婚不再是忌讳

- 命带二婚标志不意味不幸福
- 二婚往往比一婚更成熟更懂经营
- 关键看第二段婚姻的大运配合

### 三、合婚的现代观念

- 不必过度拘泥于属相冲克
- 重点关注双方用神互补和格局配合
- 感情基础和相互尊重比命理配合更重要
- 命理分析是辅助参考，不是决定因素
