# 四柱预测学讲义

- **原著**：邵伟华
- **定位**：现代四柱命理学奠基之作，当代命理复兴第一人的核心理论体系
- **特点**：以用神忌神为核心，系统化十神六亲论断法，首创男女配婚年表
- **适用**：八字基础理论、婚姻预测、配婚参考

---

## 第一章 用神与忌神

### 一、用神的定义

用神是综合平衡四柱五行的方法，定出格局高低的直接图境。取用神以生克制化为依托，对命局五行起到平衡作用。

**邵伟华用神体系的核心**：通过命局作用关系，对日干起到好作用的都为用神。

**关键突破**：
- 忌神也有发挥好作用的时候——忌神制忌神，忌神受制以吉断
- 命局虽然有用神，但弱而受制，对日干不起好作用，也以凶断
- 喜非永喜，忌非永忌——用神忌神随岁运而转化

### 二、忌神的定义

每个对日干起到坏作用的字都叫忌神。通过作用关系，忌神也能发挥用神的作用，但这取决于不同命局组合的不同结果。

### 三、用神忌神的转化实例

以命局 己酉 辛未 己亥 丁卯 为例：
- 己土生未月旺，己土旺度30%
- 丁火在局中无根生身力弱
- 辛金以年支酉金为根而泄己土
- 身弱用印比论，用神取未、己、丁，忌神为辛、酉、亥、卯
- 亥水为忌神在日支——此命二婚（忌神紧邻日干，当未土受伤应婚灾）
- 酉金为忌神冲卯木忌神——忌神制忌神应吉

## 第三章 六亲论——妻财篇

邵伟华"四柱六亲"理论中，妻财论是婚姻预测的核心。

### 一、基本原则

- 日柱之干为自己，日坐之支为配偶
- 女命以官杀为夫，官杀不混者为上
- 男女四柱看法均应同论
- 阴阳反错会招致婚灾

### 二、日支配偶宫看妻缘

**吉配置**：

| 日支坐十神 | 妻缘/夫缘 |
|-----------|----------|
| 坐正官 | 妻/夫相貌端庄，温柔贤淑，有内助之力 |
| 坐食神（无枭夺） | 妻/夫心宽体胖，贤慧持家，衣食富饶 |
| 坐财为喜 | 得妻力或得财帛；临天德者妻慈善 |
| 坐正印 | 妻/夫贤淑得力，身弱尤佳 |
| 坐财临将星 | 娶富贵名门之女 |
| 坐比劫身弱 | 妻/夫能干多助，坐比禄者尤佳 |

**凶配置**：

| 日支坐十神 | 妻缘/夫缘 |
|-----------|----------|
| 坐七杀 | 妻/夫性暴，夫妻不睦或克配偶，兼冲主凶 |
| 坐羊刃（身旺） | 妻/夫不贤凶悍，耗财甚至破产，又主克妻 |
| 坐比肩（身旺） | 妻/夫耗财，多口舌是非，重则克妻或破产 |
| 坐伤官（忌伤官） | 妻/夫破败或不得力，伤官旺好骂夫 |

### 三、财星状态看妻缘

| 财星状态 | 断语 |
|---------|------|
| 财星得禄 | 妻体健贤淑多助 |
| 财星坐贵人 | 妻貌美聪明温柔，娶富贵家之女 |
| 财星得长生 | 妻寿长可偕老 |
| 财星坐驿马 | 娶他乡之女，又主妻客死他乡 |
| 财星坐华盖 | 妻聪明而孤傲 |
| 财星坐桃花 | 由恋爱结婚，又主妻风流多艺 |
| 财临死墓绝 | 忌则克妻，喜则反吉 |
| 财旺身弱 | 不得妻力，妻多病；若羊刃反得妻助 |

### 四、特殊婚姻信息

- **正财强偏财弱**（或有正无偏）：主妻不宽妾
- **偏财强正财弱**（或有偏无正）：主妾不容妻
- **财清无刑冲克合**：主妻贤能
- **财轻无官、比劫多**或**财多身弱、无比劫**：皆主克妻
- **劫刃重、有食伤无财**：妻美则克，妻丑不克
- **日主喜财合化用财**：必得妻助；合化为忌则妻有外遇

## 第四章 男女配婚法则

### 一、配婚核心原则

1. **天合地合为最佳**：如戊辰配癸酉，甲戌配己卯，婚姻圆满持久
2. **五行相生为良配**：年命纳音相生，通常恩爱如胶
3. **天克地冲为大忌**：在所有不幸婚姻中，天克地冲最为严重
4. **用神互补为关键**：双方八字用神互补，婚姻互助

### 三、婚姻不好的标志

以下信息是明显婚姻不好的标志：
- 比肩、劫财旺，又逢生旺运
- 伤官旺无制
- 羊刃煞旺无制
- 阴差阳错日临旺相
- 日支受冲
- 天克地冲的配婚组合

### 四、婚灾化解思路

- 大运趋吉时，遇上天合地合的对象，可解性命攸关之灾
- 有缘千里来相会——想求且有机遇，方可成就
- 人的败运可在一定程度上得以调整

## 第五章 用神在婚姻预测中的应用

### 一、身旺身弱与婚姻

| 类型 | 婚姻特点 |
|------|---------|
| 身旺能胜财 | 财旺逢好运，得妻助又得钱财 |
| 身旺财旺 | 妻贤财丰，但比劫来抢则有第三者 |
| 身弱财多 | 不得妻力，妻多病，或为妻所累 |
| 身弱有印 | 印星为用，母亲或妻子如母般照顾 |
| 身弱比劫帮 | 得兄弟朋友助力，配偶也能帮扶 |

### 二、大运流年看婚灾

- 运支与日支相冲：轻则妻有灾疾，重则克妻
- 财弱行比劫运：破财伤妻
- 日支为官弱行伤官运：女命克夫
- 日支为食神行枭神运：意外伤害配偶
- 命局妻星不贤行运又不吉：离婚或私奔

### 三、实战判断流程

1. 定日元旺衰
2. 取用神忌神
3. 看配偶星（男看财、女看官杀）在何柱位
4. 看日支（婚姻宫）坐何十神
5. 看配偶星与日支的喜忌状态
6. 看大运流年是否引动婚姻宫或配偶星
7. 综合判断婚姻质量和婚期
