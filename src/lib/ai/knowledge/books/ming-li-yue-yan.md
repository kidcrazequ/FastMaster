# 命理约言精要

- **原著**：清·陈素庵著，民国·韦千里校注
- **定位**：清代命理学经典，体系严谨，方法论精湛
- **特点**：以"法""赋""论""杂论"四部分展开，看命总法最为精要
- **适用**：格局判断、用神取法、六亲论断、婚姻预测

---

## 第一章 看命总法

陈素庵"看命总法"是命理分析的第一步，也是最关键的一步。

### 一、先看月令

月令为提纲，四柱以月令为重。看命先看月令，确定日主在月令的旺衰状态。

**月令旺衰判断**：
- 日主得月令生助：如甲木生于寅卯月，为得令，日主有力
- 日主受月令克泄：如甲木生于申酉月，为失令，日主无力
- 日主与月令同气：如甲木生于亥子月，为得生，日主有根

### 二、次看日主

日主为一命之主，确定日主强弱是取用神的基础。

**日主强弱判断依据**：
1. 得令否：月令是否生助日主
2. 得地否：四支中是否有日主的根气（禄、刃、余气）
3. 得助否：天干中是否有比劫印星帮扶
4. 得势否：四柱整体五行是否有利于日主

**日主强弱五级分类**：
| 级别 | 条件 | 取用方向 |
|------|------|---------|
| 极旺 | 满盘皆比劫印星 | 从旺格，顺其旺势 |
| 偏旺 | 得令得地得助 | 抑制：用官杀、食伤、财星 |
| 中和 | 得令但不得地，或得地不得令 | 微调：小扶或小抑 |
| 偏弱 | 失令且少助 | 扶助：用印星、比劫 |
| 极弱 | 满盘皆克泄耗之字 | 从弱格，顺其弱势 |

### 三、三看用神

用神为命局之药，是决定格局高低的核心。

**用神取法七条**：
1. **扶抑用神**：日主弱则扶之以印比，日主旺则抑之以官杀食伤财
2. **调候用神**：金水生冬，需火暖之；木火生夏，需水润之
3. **通关用神**：两行对峙不和，取中间之物通之
4. **专旺用神**：从格之命，顺其旺势所取之神
5. **病药用神**：找出命局之"病"，再找"药"来治
6. **格局用神**：格局所需之物
7. **调候为急**：寒暖燥湿为先，其次扶抑

### 四、四看格局

格局决定命运层次高低。

**正格八格**：
| 格局 | 成格条件 | 破格条件 |
|------|---------|---------|
| 正官格 | 月令正官透出，有财生有印护 | 伤官克官、刑冲破害 |
| 偏官格 | 月令七杀透出，有食制有印化 | 无制无化、杀重身轻 |
| 正财格 | 月令正财透出，有官护有食生 | 比劫夺财、财多身弱 |
| 偏财格 | 月令偏财透出，有官护有食生 | 比劫夺财、财多身弱 |
| 正印格 | 月令正印透出，有官杀生印 | 财星坏印 |
| 偏印格 | 月令偏印透出，有官杀生印 | 财星坏印、枭夺食 |
| 食神格 | 月令食神透出，有财泄秀 | 枭神夺食 |
| 伤官格 | 月令伤官透出，有财泄秀 | 见官则祸（火土伤官除外） |

## 第二章 格局赋精要

### 总纲赋

> 五行生克制化之理，乃命学之根基。
> 日主为太极之点，月令为提纲之用。
> 先看月令定格局，次观四柱论喜忌。
> 用神得力格局高，忌神猖獗运途蹇。

### 正官赋要点

- 正官一位为清贵，多则为杀不为官
- 官星有根逢财生：仕途通达、配偶贤良
- 官星无根逢伤克：仕途坎坷、婚姻不顺
- 女命正官为夫星：一位清纯者为上

### 偏官赋要点

- 七杀有制方为贵，无制则为灾祸来
- 杀有食制：武贵、技术、军警之命
- 杀有印化：文贵、教育、管理之命
- 女命七杀混官：夫缘不纯，感情复杂

### 财星赋要点

- 身旺有财方为福，身弱见财反为灾
- 正财一位为正妻，偏财多为偏房、情人
- 财旺生官：夫荣妻贵
- 财弱逢劫：破财伤妻

## 第五章 六亲论——婚姻篇

### 一、男命看妻

1. 以正财为妻星：正财的位置、旺衰决定妻缘好坏
2. 以日支为妻宫：日支的字决定配偶性格和相处模式
3. 财星得力且为喜用：妻贤助力
4. 财星被克或为忌：妻缘差或克妻

### 二、女命看夫

1. 以正官为夫星：正官的位置、旺衰决定夫缘好坏
2. 以日支为夫宫：日支的字决定丈夫性格和相处模式
3. 官星清纯且为喜用：嫁贵夫
4. 官杀混杂或为忌：夫缘差或克夫

### 三、配偶远近与认识方式

| 配偶星位置 | 配偶远近 | 认识方式 |
|-----------|---------|---------|
| 在年柱 | 远方 | 长辈介绍或外地相识 |
| 在月柱 | 中等 | 工作中或朋友介绍 |
| 在日支 | 很近 | 身边人或邻居 |
| 在时柱 | 远方 | 晚辈介绍或晚年得偶 |

### 四、看婚姻时机

- **大运配偶星旺地**：适婚大运已到
- **流年配偶星出现或被合**：具体结婚年份
- **流年冲动婚姻宫**：婚姻有变化（可能结婚也可能分手）
- **红鸾天喜到位**：婚喜之年

## 第六章 杂论——婚姻相关

### 陈素庵论婚姻要诀

1. 官杀混杂之女命，宜食伤制杀或合杀留官，方能安稳
2. 比劫旺之男命，宜行财运方得良妻
3. 日支受冲者，宜择与日支相合之配偶
4. 财星入墓者，宜行冲墓之运方显妻缘
5. 伤官见官之女命，非有印制伤不可
6. 命带桃花者，宜正不宜偏，偏则招灾
7. 合婚先看年命纳音之生克，次看日支之合冲

### 韦千里校注要点

- 原书"赋"文用词古雅但含义深刻，须结合实例理解
- 看命总法为全书精华，务必烂熟于心
- 干支合冲刑害之论与婚姻关系极为密切
- 杂论部分多为实战经验总结，不可忽视
